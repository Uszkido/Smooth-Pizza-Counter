General Functions Guide

Everything you need to know to deploy, operate, and maintain the Pizza & Ice Cream Club AI system on your shop floor.

How to Run the System

▶ Option 1 — WATCHDOG (Recommended)

This is the best way to run the AI on your shop-floor PC. If the app ever crashes, it automatically restarts within 5 seconds.

# Simply double-click:
PizzaVMS\
→ WATCHDOG.bat

A black console window will open showing the system status. Do not close this window.

▶ Option 2 — Direct EXE

For a one-time manual launch, double-click the main executable directly.

PizzaVMS\
→ PizzaVMS.exe

Once you see Uvicorn running on http://0.0.0.0:8040 in the console, the system is ready.

ℹ️
After launching: Open any browser on the same PC and go to http://localhost:8040 to access the dashboard.
⚡
Auto-Start on Windows Boot

To make the AI launch automatically every time the shop-floor PC turns on or restarts, follow these steps:

Create a Shortcut of WATCHDOG.bat

Right-click on WATCHDOG.bat → select "Create Shortcut". A new file called WATCHDOG - Shortcut will appear next to it.

Open the Windows Startup Folder

Press Win + R on your keyboard. In the box that appears, type the following and press Enter:

shell:startup

A folder will open — this is Windows' auto-start folder.

Move the Shortcut into the Startup Folder

Copy the WATCHDOG - Shortcut file and paste it into the shell:startup folder you just opened.

Test It

Restart the PC. After Windows loads, the Pizza Club AI should start automatically within ~30 seconds. Open your browser and navigate to http://localhost:8040.

✅
Result: The system is now fully autonomous. No staff member needs to manually start the AI. It will be running every time the kitchen PC powers on.

Step 1 — Find the Kitchen PC's IP Address

On the kitchen PC, press Win + R, type cmd and press Enter. Then type:

ipconfig

Look for IPv4 Address under your Wi-Fi or Ethernet adapter. It will look like: 192.168.1.105

Step 2 — Recommended: Set a Static IP (Never Changing)

By default, routers can change the PC's IP on every restart. To prevent this, assign a static IP so the dashboard URL is always the same.

On the kitchen PC go to: Settings → Network → Ethernet/Wi-Fi → IP Settings → Edit → Manual

# Example static IP settings
IP Address: 192.168.1.200
Subnet Mask: 255.255.255.0
Gateway: 192.168.1.1 (your router IP)

Step 3 — Access from Any Device on the Network

On any phone, tablet, or PC on the same Wi-Fi, open a browser and type:

http://192.168.1.200:8040 (replace with your actual IP)

The Pizza Club login page will appear. Login with the Viewer account to watch the live dashboard.

⚠️
Firewall Note: If other devices can't connect, Windows Firewall may be blocking port 8040. Go to Windows Defender Firewall → Allow an app or feature and allow PizzaVMS.exe, or add an inbound rule for port 8040.

🌍 Remote Access (from OUTSIDE the shop — via Internet)

For monitoring from home or another branch over the internet, use the built-in Ngrok feature.

While the system is running, double-click START_REMOTE_ACCESS.bat — or click the "GET REMOTE LINK" button on the dashboard. A temporary https://xxxx.ngrok.io URL will be generated you can share with anyone worldwide.

Troubleshooting Quick Guide

❌ Page won't load / "Connection refused"

The app may still be starting up. Wait 30 seconds after launching and try again. If still failing, check that PizzaClub.exe is running in Task Manager under the Processes tab.

📷 Camera feed is black

Go to Settings and confirm the Camera IP. For USB cameras, the value must be 0. For IP cameras, confirm the RTSP stream URL and that the camera is powered on.

🔔 Notifications not sending

Ensure your WhatsApp phone number includes the country code (e.g. +2348012345678) and your CallMeBot API key is correct. Send a test via the Send Test Notification button in Settings.

🌐 Can't access from another device on LAN

Windows Firewall may be blocking port 8000. Go to Windows Defender Firewall → Advanced Settings → Inbound Rules → New Rule and add a rule for Port 8000.

⚡ Lots of false pizza counts

Go to Settings and increase the Confidence Threshold (e.g. from 50% to 70%). Also check the counting line position — it should cross only where real pizzas pass.

🔑 Forgot manager password

Open config.json in the application folder using Notepad. Find the "users" section and manually type in a new value for the "password" field. Save the file and restart the app.
